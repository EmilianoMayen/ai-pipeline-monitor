# Overview

The AI Pipeline Monitor platform stores information related to data pipelines, execution history, errors, AI recommendations, and system users.

The database is designed to support operational monitoring, historical analysis, AI reasoning, and future predictive capabilities.

---

# Entity Relationship Overview

The platform is composed of the following entities:

- Pipelines
- Pipeline Executions
- Errors
- AI Recommendations
- Users

---

# Pipelines

Represents every monitored data pipeline.

| Column | Type | Description |
|----------|--------|----------------|
| pipeline_id | INTEGER | Unique identifier |
| pipeline_name | TEXT | Pipeline name |
| workspace | TEXT | Workspace |
| source | TEXT | Data source |
| destination | TEXT | Destination |
| owner | TEXT | Responsible team |
| active | BOOLEAN | Active pipeline |

---

# Pipeline Executions

Stores every execution.

| Column | Type | Description |
|----------|--------|----------------|
| execution_id | INTEGER | Unique identifier |
| pipeline_id | INTEGER | Related pipeline |
| execution_date | DATETIME | Execution timestamp |
| status | TEXT | Completed / Failed / Running |
| duration_seconds | INTEGER | Execution time |
| records_processed | INTEGER | Processed records |
| error_id | INTEGER | Related error |

---

# Errors

Stores known execution errors.

| Column | Type | Description |
|----------|--------|----------------|
| error_id | INTEGER | Unique identifier |
| error_code | TEXT | Error code |
| error_message | TEXT | Error description |
| severity | TEXT | Low / Medium / High |
| root_cause | TEXT | Identified cause |

---

# AI Recommendations

Stores recommendations generated by the AI Agent.

| Column | Type | Description |
|----------|--------|----------------|
| recommendation_id | INTEGER | Unique identifier |
| execution_id | INTEGER | Related execution |
| recommendation | TEXT | Suggested action |
| confidence | FLOAT | AI confidence score |
| created_at | DATETIME | Generation timestamp |

---

# Users

Stores platform users.

| Column | Type | Description |
|----------|--------|----------------|
| user_id | INTEGER | Unique identifier |
| name | TEXT | User name |
| role | TEXT | User role |
| email | TEXT | Email address |
